# GameChanger Wheel

Spin the wheel to randomly pick an online game for game night.

**Live:** https://gamechanger-wheel.netlify.app

## Features

- **Spinning wheel** — Canvas-rendered wheel with smooth animation
- **Tick sounds** — Audio feedback as the wheel spins
- **Confetti** — Celebration when the winner is selected
- **Custom game list** — Add and remove your own games
- **PWA** — Installable on mobile and desktop

## Tech Stack

- Vanilla JavaScript
- HTML Canvas (wheel rendering)
- Web Audio API (tick sounds)
- Service Worker (offline PWA)
- Deployed on Netlify

## Development

No build step required. Open `index.html` directly or serve with any static server:

```bash
npx serve .
```

## Deployment

Deployed on Netlify as a static site.
